✔️ Este projeto é um card moderno de apresentação de produto, desenvolvido para praticar meus conhecimentos de HTML e CSS, colocando em ação conceitos como:
Tipografia com Google Fonts
Sombra e estilização de cards
Pseudoclasses (hover)
Organização de layout com Flexbox
Posicionamento absoluto da imagem
Cores, tamanhos e interações
Estética e atenção aos detalhes.
O projeto foi inspirado em um tutorial do YouTube, mas totalmente personalizado por mim para treinar criatividade e estilo próprio.

✔️ Tecnologias usadas: 
HTML5
CSS3
Google Fonts
SVG Icons (carrinho)

✔️O objetivo deste mini-projeto foi praticar:
Construção de interfaces modernas
Responsividade básica
Efeitos com hover e transições
Posicionamento de elementos (layout + imagem flutuante)
